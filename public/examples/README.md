# Examples

- **`capabilityhostprotocol.com.txt`** — a **real, live** `capabilities.txt`
  (the CHP adapter ecosystem). This is an actual published file.
- **`illustrative/`** — **illustrative** examples showing how different markets
  would declare capabilities. The domains are fictional (`*.example`); they
  demonstrate the format, not live sites. Use them as starting templates.

Real published files belong in [`../adopters.md`](../adopters.md).
